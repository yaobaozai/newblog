# 项目实战：电话号码验证器

  • 题目链接 (opens new window)

# 思路讲解

  • 这个题目考察正则表达式，不要被题目所吓倒。
  1. 有可能是1开头，^1?
  2. 每个数字与数字之间可能有空格，\s?
  3. 后三个数字有可能有括号也有可能没括号 (\d{3}|\(\d{3}\))
  4. 后续数字可能会按-或空格分隔 [-\s]? []表示联合条件，在这里就是-加上空格都可以
  5. 后续三个数字，以及可能会按-或空格分隔 \d{3}[-\s]?
  6. 结尾后4个数字 \d{4}$

# 代码实现

function telephoneCheck(str) {
  if (str.match(/^1?\s?(\d{3}|\(\d{3}\))[-\s]?\d{3}[-\s]?\d{4}$/)) {
    return true;
  } else {
    return false;
  }
}
